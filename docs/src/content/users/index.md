# How to Play Pokemon Duel

Welcome to Pokemon Duel! This guide will help you get started.

## What is Pokemon Duel?

Pokemon Duel is a multiplayer real-time web board game where 2-6 players compete in strategic Pokemon battles.

## Getting Started

### Creating a Room

1. Click "Create Room" on the home screen
2. Share the room code with your friends
3. Wait for players to join (2-6 players)
4. Click "Start Game" when everyone is ready

### Joining a Room

1. Get a room code from your friend
2. Enter the code on the home screen
3. Click "Join Room"
